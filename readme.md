# Guns.lol View Bot (Playwright Stealth Edition)

A high-performance, asynchronous view bot for Guns.lol profiles using Playwright. This tool simulates real human behavior to bypass basic bot detection and uses SOCKS5 proxies to scale views.

## 🚀 Features

- **Asynchronous Execution:** Handles multiple browser instances simultaneously.
- **Stealth Mode:** Uses `--disable-blink-features=AutomationControlled` to hide automation flags.
- **Human Simulation:** Simulates smooth mouse movements, scrolling, and random clicks.
- **Proxy Support:** Designed specifically for SOCKS5 proxies.
- **Live Dashboard:** Real-time stats showing Success/Fail counts and views per minute.

## 🛠 Prerequisites

- Python 3.8+
- 16GB+ RAM (32GB recommended for high thread counts)
